Organização dos Exercícios em C++

Este repositório reúne uma coleção de exercícios desenvolvidos em C++ para a disciplina de Algoritmos e Programação de Computadores. A estrutura foi planejada para manter os arquivos bem organizados, facilitando tanto a navegação quanto os testes de cada programa.

Descrição dos diretórios

Cada diretório corresponde a um grupo de exercícios relacionados a um tema específico da linguagem C++. A organização interna segue um padrão simples e uniforme.

Arquivos .cpp

Cada arquivo .cpp representa um exercício individual. Os códigos seguem a estrutura tradicional de qualquer programa básico em C++: leitura de dados, processamento e exibição dos resultados.
Os assuntos trabalhados incluem:

lógica de programação

repetição

vetores e matrizes

modularização com funções

structs e manipulação de dados
