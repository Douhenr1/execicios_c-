Descrição dos diretórios

Cada diretório contém um conjunto de exercícios voltados para um tópico específico da linguagem C++. A estrutura interna é sempre a mesma, garantindo uniformidade entre todas as pastas.

Arquivos .cpp
Cada arquivo representa um exercício individual. Os programas seguem um padrão simples, envolvendo leitura de dados, processamento e saída dos resultados. Os temas incluem lógica básica, laços de repetição, vetores, matrizes, modularização e uso de structs.

Subpasta Executaveis/
Dentro dela ficam armazenados os arquivos executáveis (.exe) correspondentes aos exercícios. Cada executável é gerado a partir do código-fonte de mesmo nome, mantendo organização e fácil identificação.

Como executar

Abra o projeto no Visual Studio Code ou outro ambiente C++ de sua preferência.

Acesse a pasta do exercício que deseja rodar.

Compile o código usando o compilador g++:

g++ EX01.cpp -o Executaveis/EX01.exe


Execute o programa:

./Executaveis/EX01.exe


Observações

Todas as pastas seguem a mesma estrutura e funcionam de forma independente.

A nomenclatura dos arquivos é padronizada (EX0X.cpp e EX0X.exe).

A organização busca facilitar a prática e o estudo modular da linguagem C++.

Se quiser, posso montar também uma versão mais formal, mais simples ou até transformar isso em um README.md profissional.
